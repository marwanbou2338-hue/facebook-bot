# 🤖 بوت ماسنجر المجموعات

بوت ماسنجر متقدم للمجموعات يعتمد على `appstate` للمصادقة.

---

## 🚀 طريقة التشغيل

### 1. تثبيت المتطلبات
```bash
cd bot
npm install
```

### 2. الحصول على ملف appstate
- ثبّت إضافة **c3c-fbstate** أو **Get FB State** على متصفح Chrome
- سجّل دخولك على Facebook
- استخدم الإضافة لتصدير الـ appstate
- احفظ الملف باسم `appstate.json` داخل مجلد `bot/`

### 3. إعداد ملف config.json
افتح `bot/config.json` وعدّل الحقول التالية:
```json
{
  "superAdminId": "123456789",  ← ضع ID الفيسبوك الخاص بك هنا
  ...
}
```

### 4. تشغيل البوت
```bash
npm start
```

## 📋 قائمة الأوامر

### الأوامر العامة
| الأمر | الوصف |
|-------|--------|
| `الاوامر` | عرض قائمة الأوامر |
| `سيرفر` أو `ابتيم` | عرض حالة البوت ووقت التشغيل |

### أوامر المحرك (للأدمن فقط)
| الأمر | الوصف |
|-------|--------|
| `محرك تشغيل` | تشغيل المحرك في المجموعة الحالية |
| `محرك إيقاف` | إيقاف المحرك في المجموعة الحالية |
| `محرك رسالة [النص]` | تغيير نص رسالة المحرك |
| `محرك وقت [ثواني]` | تغيير الفاصل الزمني بالثواني |
| `محرك ذكي تشغيل` | الإرسال فقط عند وجود نشاط |
| `محرك ذكي إيقاف` | إيقاف الوضع الذكي |

### أوامر القفل (للأدمن فقط)
| الأمر | الوصف |
|-------|--------|
| `قفل تشغيل` | قفل المجموعة الحالية |
| `قفل إيقاف` | فك قفل المجموعة الحالية |
| `قفل ذكي تشغيل` | قفل جميع المجموعات |
| `قفل ذكي إيقاف` | إيقاف القفل الذكي |

### أوامر الإدارة (للسوبر أدمن فقط)
| الأمر | الوصف |
|-------|--------|
| `رفع [ID]` | منح صلاحيات أدمن البوت |
| `خفض [ID]` | إزالة صلاحيات أدمن البوت |

> **ملاحظة:** يمكن استخدام أوامر `رفع` و`خفض` بالرد على رسالة المستخدم بدون كتابة الـ ID

## 📁 هيكل المشروع
```
bot/
├── index.js              # نقطة الدخول الرئيسية
├── config.json           # الإعدادات (superAdminId، المحرك، القفل)
├── appstate.json         # ملف المصادقة (أضفه يدوياً)
├── package.json
├── commands/
│   ├── help.js           # أمر: الاوامر
│   ├── status.js         # أمر: سيرفر / ابتيم
│   ├── admin.js          # أوامر: رفع / خفض
│   ├── engine.js         # أوامر المحرك
│   └── lock.js           # أوامر القفل
├── systems/
│   ├── engine.js         # منطق نظام المحرك
│   └── lock.js           # منطق نظام القفل
└── utils/
    ├── config.js         # إدارة ملف الإعدادات
    └── permissions.js    # إدارة الصلاحيات
```

## ⚙️ الإعدادات (config.json)

```json
{
  "superAdminId": "ID_هنا",
  "botAdmins": [],
  "engine": {
    "message": "نص الرسالة الدورية",
    "interval": 60000,
    "smartMode": false,
    "activeGroups": {}
  },
  "lock": {
    "smartLock": false,
    "lockedGroups": []
  }
}
```

| الحقل | الوصف |
|-------|--------|
| `superAdminId` | ID الفيسبوك للمطور (سوبر أدمن) |
| `botAdmins` | قائمة IDs أدمن البوت |
| `engine.message` | نص الرسالة الدورية |
| `engine.interval` | الفاصل الزمني بالمللي ثانية |
| `engine.smartMode` | الوضع الذكي (إرسال عند النشاط فقط) |
| `lock.smartLock` | قفل ذكي (جميع المجموعات) |
| `lock.lockedGroups` | قائمة المجموعات المقفلة |
